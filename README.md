# aott33.github.io

Personal technical blog built with Jekyll and the Chirpy theme.

## About

Learning in public by documenting hands-on projects in automation, SCADA, IIoT, and Kubernetes homelab builds.

## Tech Stack

- Jekyll 4.x
- Chirpy Theme
- GitHub Pages
- GitHub Actions for CI/CD

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000`
